To create the System Report task, follow the instructions in the vFabric Application Director user guide section "Add a Custom Task to the Catalog" in chapter 8.

Use the following values
------------------------
Name: SystemReport
Version: 1.0
Description: A script to generate a system report to show various system information 
Supported OSes: CentOS32 5.6.0, CentOS32 5.7.0, CentOS32 6.0.0, CentOS64 5.6.0, CentOS64 6.0.0, SLES32 11.1.0, SELS64 11.1.0, RHEL32 6.1.0, RHEL64 6.1.0  

Use the contents of script.sh for the action script.

This task can then be added to a deployment profile in the execution flow by following the instruction in the vFabric Application Director user guide section "Add Custom Tasks" in chapter 11.
