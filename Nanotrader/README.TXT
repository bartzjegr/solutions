To create and deploy Nanotrader application blueprint, follow the steps in the Nanotrader-Blueprint-1.0.0.pdf.

Before starting to work on the blueprint, download the following files 

* DataGenerator.zip
* nanotrader-template.tar
* spring-nanotrader-asynch-services-0.0.1.BUILD-SNAPSHOT.war
* spring-nanotrader-mobile-0.0.1.BUILD-SNAPSHOT.war
* spring-nanotrader-services-0.0.1.BUILD-SNAPSHOT.war
* spring-nanotrader-web-0.0.1.BUILD-SNAPSHOT.war

from Github 
https://github.com/vmware-applicationdirector/solutions.git
git@github.com:vmware-applicationdirector/solutions.git

Host the downloaded files in a http file server that is accessible to VMs 
deployed by vFabric Application Director in vCloud Director.
